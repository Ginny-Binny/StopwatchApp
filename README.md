# Stopwatch App

Welcome to the Stopwatch App! This simple application allows you to track time intervals with precision and ease. Whether you need to time your workouts, monitor cooking durations, or track any other timed activities, this app has got you covered.

## Features

- **Start, Pause, and Reset**: Start the stopwatch to begin timing, pause it to temporarily halt the timer, and reset it to zero when needed.
- **Accurate Timekeeping**: The stopwatch app provides precise timing to ensure accurate tracking of time intervals.
- **Lap Functionality**: Record lap times to track multiple intervals within a session. Each lap time is displayed alongside the total elapsed time.
- **User-Friendly Interface**: The app features a clean and intuitive interface, making it easy to navigate and operate.

## Getting Started

To get started with the Stopwatch App, follow these steps:

1. Clone the repository to your local machine using `git clone https://github.com/Ginny-Binny/StopwatchApp`.
2. Open the project in your preferred code editor.
3. Launch the app by opening the `app.js` file in any code editor.

## Usage

- Press the "Start" button to initiate the stopwatch.
- Press the "Pause" button to temporarily stop the timer without resetting it.
- Use the "Reset" button to reset the stopwatch to zero.
- To record lap times, click on the "Lap" button. Each lap time will be displayed below the stopwatch.
